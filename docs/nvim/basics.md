## nvim とは

**Neovim (nvim)** = モダン Vim エディタ

```
Vim の互換性を保ちながら、プラグイン機能などを大幅に改善
→ 軽量で高速、カスタマイズ性が高い
```

### Vim 基本モード

| モード | 説明 | 入れ方 |
|--------|------|--------|
| **Normal** | コマンド実行（移動、削除等） | Esc キー |
| **Insert** | テキスト入力 | i, a, o など |
| **Visual** | テキスト選択 | v (character), V (line), Ctrl+V (block) |
| **Command** | コマンド実行（保存、終了等） | : で開始 |

## 起動・終了

### 起動

```bash
# カレントディレクトリを開く
nvim .

# 特定ファイルを開く
nvim /path/to/file.txt

# 複数ファイルを開く
nvim file1.txt file2.txt
```

### 終了

| 操作 | コマンド | 説明 |
|------|---------|------|
| **保存して終了** | `:wq` または `ZZ` | 変更を保存して終了 |
| **終了（保存なし）** | `:q!` | 変更を破棄して終了 |
| **保存のみ** | `:w` | ファイルを保存（終了しない） |

## 基本操作

### モード切り替え

| キー | 操作 |
|------|------|
| **Esc** | Normal モードに戻る |
| **i** | カーソル前で Insert モード |
| **a** | カーソル後で Insert モード |
| **o** | 新規行を下に作成して Insert モード |
| **O** | 新規行を上に作成して Insert モード |
| **v** | Visual モード（文字単位） |
| **V** | Visual モード（行単位） |
| **Ctrl+V** | Visual モード（ブロック単位） |
| **:command** | Command モード |

### 移動（Normal モード）

| キー | 移動先 |
|------|--------|
| **h** | ← 左 |
| **j** | ↓ 下 |
| **k** | ↑ 上 |
| **l** | → 右 |
| **0** | 行の最初 |
| **$** | 行の最後 |
| **w** | 次の単語 |
| **b** | 前の単語 |
| **^** | 行の最初（スペース後） |
| **G** | ファイルの最後 |
| **gg** | ファイルの最初 |
| **Ctrl+U** | 半ページ上スクロール |
| **Ctrl+D** | 半ページ下スクロール |

### 編集（Normal モード）

| キー | 操作 |
|------|------|
| **x** | カーソルの文字削除 |
| **dd** | 行全体削除 |
| **d[motion]** | モーション範囲削除（例: dw = 単語削除） |
| **yy** | 行全体コピー |
| **y[motion]** | モーション範囲コピー |
| **p** | ペースト（カーソル後） |
| **P** | ペースト（カーソル前） |
| **u** | アンドゥ |
| **Ctrl+R** | リドゥ |
| **~** | 大文字小文字切り替え |

### 実例

```
ファイル内容:
line 1: This is a test
line 2: Hello World
line 3: Goodbye

操作:
Normal モードで gg          → ファイル最初に移動
j                          → 下に移動（line 2）
w                          → 単語を進める（Hello）
dw                         → Hello を削除
p                          → ペースト（復元）
yy                         → 行全体コピー（line 2）
j                          → 下に移動（line 3）
p                          → ペースト

結果:
line 1: This is a test
line 2: Hello World
line 3: Goodbye
line 4: Hello World        （コピーされた）
```

## ファイルエクスプローラ

### Netrw（デフォルトファイルエクスプローラ）

```bash
# ファイルエクスプローラ起動
:E          # 現在のディレクトリを開く
:e .        # 現在のディレクトリを開く（別形式）
:Explore    # 専用ウィンドウで開く
```

### 操作方法

| キー | 操作 |
|------|------|
| **Enter** | ファイルを開く / ディレクトリに移動 |
| **-** | 親ディレクトリに戻る |
| **d** | 新規ディレクトリ作成 |
| **%** | 新規ファイル作成 |
| **D** | ファイル/ディレクトリ削除 |
| **R** | ファイル/ディレクトリ名変更 |
| **v** | ファイルを垂直分割で開く |
| **s** | ファイルを水平分割で開く |
| **t** | ファイルを新規タブで開く |
| **i** | 表示形式を変更（詳細 ↔ シンプル） |
| **q** | ファイルエクスプローラを閉じる |

### 実例

```bash
# 1. ファイルエクスプローラを開く
nvim .

# 2. ファイル一覧が表示される:
# ./
# ../
# src/
# package.json
# README.md

# 3. src/ に移動
# カーソルを src/ に当てて Enter

# 4. src/ 内のファイル一覧を表示

# 5. main.js を開く
# カーソルを main.js に当てて Enter

# 6. 親ディレクトリに戻る
# - キーを押す
```

### ファイルエクスプローラウィンドウの分割

```bash
# 1. ファイルエクスプローラを垂直分割で開く
:E

# 2. 新規ファイルを垂直分割で開く
# ファイル上で v を押す

# 3. 新規ファイルを水平分割で開く
# ファイル上で s を押す

# 4. ウィンドウ間を移動
Ctrl+W Ctrl+W  # 次のウィンドウ
Ctrl+W h       # 左のウィンドウ
Ctrl+W j       # 下のウィンドウ
Ctrl+W k       # 上のウィンドウ
Ctrl+W l       # 右のウィンドウ
```

## 編集モード

### Insert モード

```bash
# Insert モード開始
i       # カーソル前で入力
a       # カーソル後で入力
I       # 行の最初で入力
A       # 行の最後で入力
o       # 新規行下で入力
O       # 新規行上で入力

# テキスト入力
<text here>

# Insert モード終了
Esc
```

### Visual モード（選択）

```bash
# Visual モード開始
v       # 文字単位
V       # 行単位
Ctrl+V  # ブロック単位

# 選択を拡張
hjkl / w / b / $ / 0 など

# 操作（選択状態で）
d       # 削除
y       # コピー
c       # 削除して Insert モード
>       # インデント
<       # アンインデント
```

### 実例

```
ファイル:
function hello(name) {
    console.log("Hello " + name);
}

操作:
V               → line 1 を選択（Visual line）
j               → line 2 も選択
y               → コピー
G               → ファイル最後に移動
o               → 新規行
p               → ペースト

結果:
function hello(name) {
    console.log("Hello " + name);
}
function hello(name) {
    console.log("Hello " + name);
}
                ← 新しい行（貼り付け）
```

## 検索・置換

### 検索

```bash
# 検索開始
/pattern    # 次を検索
?pattern    # 前を検索

# 検索結果を移動
n           # 次の一致
N           # 前の一致

# 検索をキャンセル
Esc
```

### 置換

```bash
# 現在の行を置換
:s/old/new/         # 最初の1つを置換
:s/old/new/g        # 全て置換

# 全行を置換
:%s/old/new/g       # ファイル全体で置換

# 選択範囲を置換（Visual モード後）
:s/old/new/g
```

### 実例

```
ファイル:
Hello World
Hello Everyone
Goodbye World

操作:
:/World             → World を検索
n                   → 次の World に移動
:%s/World/Universe/g  → World を全て Universe に置換

結果:
Hello Universe
Hello Everyone
Goodbye Universe
```

## 実践例

### シナリオ 1: ファイルを開いて編集

```bash
# 1. ファイルエクスプローラを開く
nvim .

# 2. src/ に移動して Enter

# 3. app.js を開く

# 4. Insert モードで編集
# 適当な行で i を押して入力

# 5. 保存して終了
Esc
:wq
```

### シナリオ 2: 複数ファイルを同時編集

```bash
# 1. ファイルエクスプローラを開く
nvim .

# 2. file1.js を開く
# Enter で開く

# 3. ウィンドウ分割
# file1.js の上で :vsplit を実行
:vsplit

# 4. file2.js を開く
# ファイルエクスプローラで v を押す

# 5. ウィンドウ間を移動
Ctrl+W l  # 右ウィンドウに移動
Ctrl+W h  # 左ウィンドウに移動

# 6. 保存
:w

# 7. 終了
:q
```

### シナリオ 3: テキスト置換

```bash
# 1. ファイルを開く
nvim app.js

# 2. 全ての var を const に置換
:%s/var /const /g

# 3. 確認して保存
# 置換内容を確認
:set hlsearch   # ハイライト有効

# 4. 保存
:w

# 5. 終了
:q
```

## トラブルシューティング

### Q: Insert モードから抜けられない

```bash
# Esc キーを何度も押す
Esc Esc Esc
```

### Q: ファイルが保存されない

```bash
# 保存コマンドを実行
:w

# 強制保存
:w!
```

### Q: ファイルエクスプローラが開かない

```bash
# 別の方法で開く
:Explore
:E
:edit .
```

### Q: 編集内容を取り消したい

```bash
# Undo
u

# Redo
Ctrl+R

# 複数回 Undo
uu u (3回 u キー)
```
